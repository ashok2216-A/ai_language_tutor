# AI Language Tutor MVP

## Features
- Diagnostic Test
- Speaking Practice Module
- Writing Assistant
- Adaptive Quizzes
- Progress Dashboard

## Setup
1. Clone the repository
2. Open `index.html` in a modern browser
3. No additional setup required for MVP
